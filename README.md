# ThiqaVR

ThiqaVR is a Virtual Reality Exposure Therapy (VRET) system designed to help individuals overcome public speaking anxiety through immersive VR environments and heart-rate monitoring.

## Project Overview

Public speaking anxiety affects many individuals and can impact academic, professional, and personal performance. ThiqaVR provides a safe and controlled virtual environment where users can gradually improve their confidence while practicing public speaking.

The system monitors the user's heart rate and adapts the difficulty level of the virtual audience based on stress levels, creating a personalized exposure therapy experience.

## Features

- Virtual Reality public speaking environments
- Real-time heart-rate monitoring
- Adaptive audience difficulty adjustment
- Performance tracking and data collection
- Interactive and immersive user experience
