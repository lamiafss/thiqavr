# ThiqaVR

## Graduation Project

ThiqaVR is a Bachelor's Graduation Project developed by Computer Science students at Imam Mohammad Ibn Saud Islamic University (IMSIU).

The project is a Virtual Reality Exposure Therapy (VRET) system designed to help individuals overcome public speaking anxiety through immersive virtual environments and real-time heart-rate monitoring.

## Project Overview

Public speaking anxiety is one of the most common forms of social anxiety and can negatively affect academic, professional, and personal performance.

ThiqaVR provides users with a safe and controlled virtual environment where they can practice public speaking in front of different audiences. The system continuously monitors the user's heart rate and adapts the difficulty level based on stress levels, creating a personalized exposure therapy experience.

## Objectives

- Help users reduce public speaking anxiety.
- Provide immersive VR exposure therapy sessions.
- Monitor physiological responses using heart-rate data.
- Adapt difficulty levels dynamically based on user stress.
- Track user progress throughout therapy sessions.

## Features

- Virtual Reality public speaking environments.
- Real-time heart-rate monitoring.
- Adaptive audience difficulty adjustment.
- Performance tracking and data collection.
- Interactive and immersive user experience.
- Personalized exposure therapy approach.
